Discover subnets from already added routers or by adding new routers. Specify the router, switch, and gateway server in your network, on the Add Routers screen, and schedule scanning. On adding routers, OpUtils scans its ARP table using SNMP and CLI queries, to discover subnets. It also identifies MAC addresses by querying IPRoute table and LipRoute table.
